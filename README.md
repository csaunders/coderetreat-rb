# Coderetreat.rb

A skeleton project to get you up and running for Global Day of Code Retreat using
the Ruby language.

**Prerequisites**

- A ruby manager such as [chruby](https://github.com/postmodern/chruby#install)
- Bundler: `gem install bundler`

**Setup**

Install your dependencies:

`bundle install`

**Run Guard and Start Hacking**

`bundle exec guard`

**After 45 Minutes throw everything away**

`git checkout . && git clean -f`
